Performance Indices:

_________________

 

Queue Length:

Number of customers at a center, either waiting or receiving service.

 

Queue Time:

Average time spent by the class in a station queue. It does not include the Service Time, but only the time spent in the queue.

 

Residence Time:

Total time spent at a station by a customer, both queueing and receiving service.

 

Response Time

Average time spent in a station by a customer for a single request.

 

System Response Time:

It's the summary of every response time for all stations belonging to the system.

It corresponds to our intuitive notion of response time: the interval between a user submits a request and receives a response.

 

Utilization:

Represents the utilization percentage of a resource, from a theoretical minimum of 0 (0%) to a theoretical maximum of 1 (100%). May be greater than 1 if a station has more than one server.

 

Throughput:

It's the rate at which customers departs from a station.

 

System Throughput:

It's the rate at which customers departs from the system.

 

Customer Number:

This index represents the average number of customers in the system.

If the index is associated to a closed class, then it is equal to the number of customers in the class. This is also true if there aren't any errors or drops in the system.

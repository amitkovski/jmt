Define Performance indices

_________________

 

Performance Indices are the output of the model; you can choose the ones you want to show in the output from below:

  • Queue Length

  • Queue Time

  • Residence Time

  • Response Time

  • System Response Time

  • Utilization

  • Throughput

  • System Throughput

  • System Response Time

  • Customer Number

 

For each chosen index you must indicate the associated class and station, the Confidence Interval and the Max Relative Error (these two final parameters are numbers included in the interval >= 0.1 and <=0.9).

When you press "Performance Indices" from Define menu, JSIM will show this panel:

 

 

1 - Select the index you want to add to the model from this menu:

 

 

2 - Press and the index will be added to the panel. Then you must set the parameters.

 

3 - Insert from this menu the type of class you want to associate. You can also select "all classes":

 

 

4 - Select the Station for the index from Station menu:

 

 

5 - Insert the number of Confidence Interval and Max Relative Error in these columns:

 

 

    Confidence Interval represents the confidence interval of solution.

    Max Relative Error is the greatest sample error.

     

6 - Repeat these steps for all the indices you want to include in the model output.

 

NOTE: if you make some errors in this part, JSIM won't show any message, but a warning or an error message will be showed when you start the simulation.

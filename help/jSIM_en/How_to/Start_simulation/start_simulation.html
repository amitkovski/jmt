Start Simulation

__________

 

Whenever a model has been defined you are ready to start the simulation.

To see more about simulation parameters see here.

If the model contains errors the simulation won't start and an error window will appear, see errors & warnings.

After the simulation has started correctly a new window will appear:

 

Under the windows there are the usual buttons to control the simulation. It's also available a status bar reporting current progress of simulation.

 

In case of errors, at the end of the simulation the green 'V' for every index per station won't appear and either a yellow '?' (question mark) or a red 'X' will take its place.

 

Usually this is due by some semantic error in the model, not a syntactic one. For example if you define a performance index for a station which has no users, the simulation will start but no index will be calculated.

In this way the model is syntactically correct (so simulation does start) but semantic is quite unclear (so a question mark or a 'x' will appear in place of simulation results for that index).

 

For more information see simulation results or error & warnings.

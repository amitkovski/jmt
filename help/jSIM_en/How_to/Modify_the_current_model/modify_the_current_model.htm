Modify the current model

_________________

 

1 - Whenever a model has been created and/or a simulation took place, you can always step back to modify the objects of the current model.

To modify the current model parameters after a simulation just close the simulation windows with the button and select the tab you're interested in modifying (this time is no more required to follow an exact order):

 

 

If no simulation took place just select the tab you want to modify.

 

In this way you can change the current settings in the same way you created the model the first time.

(To find more information on every tab, just click on its name on the picture)

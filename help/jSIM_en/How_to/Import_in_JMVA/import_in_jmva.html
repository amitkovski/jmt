Import in JMVA

_________________

 

Whenever a model has been created and/or simulated is possible to automatically import the same model into JMVA tool of JMT.

 

To do it just select Action->Import in JMVA... menu command and the model will be imported into JMVA.

 

Eventually a windows will appear with information about errors in the model.

 

After have correctly executed the action a new JMVA windows will open and from now on you will be able to run JMVA with the model you created with JSIM.

 

 

Please refers to JMVA user's guide for more help on JMVA functionalities.
